### 3.0版本主要特性 Characteristic

#### &hearts; 可视化表单设计器

- 基于JQ UI的元素拖拽  `Draggable`   `Sortable`
- 元素可移动拖拽设计
- 界面布局自定义化

#### &hearts; 动态表单构建器

- 全新动态构建器
- 无实体页面及数据，全部由表单构建器进行管理

#### &hearts; 脚本定义支持
-  `load_satr_fun` 页面构建前执行的函数
- `load_end_fun` 页面构建完成后函数
- 内置常见的所有脚本 `Ajax Upload` 支持jQ所有脚本

#### &hearts; 函数定义支持
- 支持使用 `commonfun.sFun` 进行数据交互，使得表单动态而又活起来。

####  &hearts; 设计版本管理
- 多版本设计支持

#### &hearts; 元素管理支持
- 支持对表单函数的管理
